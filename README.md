# Рух товарів — веб-аналізатор

## Встановлення

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python app.py
```

Відкрити в браузері: http://localhost:5000

## Можливості

- Завантаження XLS-файлів формату EPI (Рух товарів кількісний) drag & drop
- Автоматична шапка: Магазин, Склад, Період
- Зведена таблиця по місяцях для кожного артикулу
- Колонки: ПрВ, Кнк, ПрИ, СпП, Апс, Залишок, Ціна, Сума
- Підсумки по кожному артикулу + загальний підсумок
- Пошук по таблиці
- Експорт у Excel (.xlsx) зі стилями

## Формула залишку

```
Залишок = (ПрВ + СпП + ПрИ) − Кнк − Апс
(СпП та ПрИ враховуються зі знаком — від'ємні у файлі)
```
